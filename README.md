# Task Management System in Python

## Overview

A simple command-line Task Management System built using Python. This project allows users to manage their daily tasks efficiently through basic CRUD operations (Create, Read, Update, Delete).

## Features

* Add multiple tasks
* View all tasks
* Update existing tasks
* Delete tasks
* User-friendly menu-driven interface
* Continuous execution until user exits

## Technologies Used

* Python 3
* Lists
* Loops
* Conditional Statements
* Functions

## How to Run

1. Install Python 3.
2. Download or clone the repository.
3. Open terminal/command prompt.
4. Run:

python task_management.py

## Sample Operations

1. Add Task
2. Update Task
3. Delete Task
4. View Tasks
5. Exit
